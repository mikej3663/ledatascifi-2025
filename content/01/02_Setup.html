Step 9: Adding superpowers to JupyterLab¬∂

These extensions will give you spellchecking, code hints, and automatic formatting for your code üòç.

  1. Run these in terminal (Mac) or Anaconda Powershell Prompt (Windows) one at a time:

    pip install jupyterlab-spellchecker

    • Mac users: To install the spellchecker, your TA had to install node.js in order to install the spell checker extension. MacOS users who also installed homebrew can install this in terminal via brew install node.

  2. Now run jupyter lab in the terminal window to open Jupterlab.

  3. Click on settings, then Advanced Settings Editor. Go to the keyboard shortcuts tab (you can use the search bar in the middle of the screen). Then click on ‚ÄúJSON Settings Editor‚Äù (in the upper right of the main area) and it will open a user preference box.

    Note Mac users might prefer to replace ‚ÄúCtrl‚Äù with ‚ÄúCmd‚Äù in the code below.
